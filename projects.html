E-commerce Grocery Store

The mini e-commerce grocery shop project enables users to conveniently browse, select, and purchase groceries online. Key features include user authentication, a user-friendly product catalog, a responsive shopping cart, secure checkout, and order history. Technologies include HTML, CSS, JavaScript, and a backend framework with payment integration. The project enhances the online grocery shopping experience by offering a seamless and responsive platform for users.
